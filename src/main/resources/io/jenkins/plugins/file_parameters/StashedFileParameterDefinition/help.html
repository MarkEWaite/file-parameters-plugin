File parameter compatible with Pipeline but this one stash the file, how to use it in a declarative pipeline:

pipeline {
  agent any
  parameters {
    stashedBase64File(name: 'FILE-STASH')
  }
  stages {
    stage('Example') {
      steps {
        unstash "FILE-STASH"
        echo(/loaded '${readFile("./FILE-STASH")}'/)
      }
    }
  }
}
