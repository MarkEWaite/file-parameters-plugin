File parameter compatible with Pipeline but this one stash the file and must be used to handle bug files, how to use it in a declarative pipeline:

pipeline {
  agent any
  parameters {
    stashedFile(name: 'FILE-STASH')
  }
  stages {
    stage('Example') {
      steps {
        unstash "FILE-STASH"
        sh 'cat FILE-STASH'
      }
    }
  }
}
